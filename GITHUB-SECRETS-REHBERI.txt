GitHub Secrets Rehberi

Asagidaki secret'lari GitHub repo ayarlarinda Actions secrets olarak ekle:

1. BUILD_CERTIFICATE_BASE64
   Aciklama: .p12 sertifikanin Base64 degeri

2. P12_PASSWORD
   Aciklama: .p12 export sifresi

3. BUILD_PROVISION_PROFILE_BASE64
   Aciklama: .mobileprovision dosyasinin Base64 degeri

4. KEYCHAIN_PASSWORD
   Aciklama: CI icinde gecici keychain parolasi

5. IOS_TEAM_ID
   Aciklama: Apple Developer Team ID

6. APPSTORE_CONNECT_KEY_ID
   Aciklama: App Store Connect API Key ID

7. APPSTORE_CONNECT_ISSUER_ID
   Aciklama: App Store Connect Issuer ID

8. APPSTORE_CONNECT_API_KEY_BASE64
   Aciklama: AuthKey_XXXXXX.p8 dosyasinin Base64 degeri

Base64 uretme komutlari:

.p12
[Convert]::ToBase64String([IO.File]::ReadAllBytes("C:\path\certificate.p12"))

.mobileprovision
[Convert]::ToBase64String([IO.File]::ReadAllBytes("C:\path\profile.mobileprovision"))

.p8
[Convert]::ToBase64String([IO.File]::ReadAllBytes("C:\path\AuthKey_ABC123XYZ.p8"))

GitHub'da ekleme yolu:
Repo > Settings > Secrets and variables > Actions > New repository secret
